# Quickly start

```
$ git clone https://github.com/cmirnow/slider-on-rails.git
$ cd slider-on-rails
$ bundle install
$ bin/rails db:create
$ bin/rails db:migrate
$ bin/rails db:seed
$ yarn install --check-files
$ bin/rails s
```

## Running specs

```
$ bundle exec rspec
```
or

```
$ bundle exec rspec -fd

```
